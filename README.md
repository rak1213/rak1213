## What I'm Building
 
Currently the founding engineer at a VC-backed startup where I own the full stack — from FastAPI backend services and event-driven architecture on AWS/GCP to multi-agent orchestration with LangGraph, production RAG pipelines, and closed-loop evaluation infrastructure.
 
**Some of what that involves:**
 
- Custom agent harness built from scratch (Pydantic AI SDK) with conversation compaction, task decomposition, and auto tool calling
- Multi-provider LLM orchestration with capability-based routing across OpenAI, Anthropic, and Google with automatic failover
- Async data ingestion pipeline processing 10+ file formats from HubSpot, Drive, Gmail, Slack via Composio
- Agentic RAG with multimodal embeddings, cross-encoder reranking, multivector retrieval — Precision@10 above 92%
- CI-gated prompt deployment with regression evaluation, model-version pinning, and LLM-as-judge scoring
- Full observability stack with LangSmith tracing, Azure monitoring, and automated feedback-to-few-shot curation loops
